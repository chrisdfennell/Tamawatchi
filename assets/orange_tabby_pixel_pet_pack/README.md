# Orange Tabby Garmin Pixel Pet Pack

Classic 16-bit virtual pet asset pack for small Garmin MIP/AMOLED screens.

- Native transparent PNGs are in `native/`.
- 4x nearest-neighbor reference PNGs are in `upscaled_4x/`.
- Pet poses are 48x48, food/toys/icons are 24x24, bubbles/effects are 32x32, backgrounds are 240x240.
- Sprite sheet order: idle, happy, sad, hungry, sleeping, eating, playing, sick_poopy, egg, baby, teen, adult.
