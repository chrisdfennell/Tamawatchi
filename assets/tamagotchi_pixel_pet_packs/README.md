# Tamagotchi Pixel Pet Packs

Complete Garmin-friendly 16-bit virtual pet packs. Each folder contains native transparent PNGs and 4x nearest-neighbor reference PNGs.

- `orange_tabby_pixel_pet_pack/` - Fluffy Orange Tabby Cat (43 native PNGs + 43 4x PNGs)
- `golden_retriever_puppy_pixel_pet_pack/` - Playful Golden Retriever Puppy (43 native PNGs + 43 4x PNGs)
- `baby_red_dragon_pixel_pet_pack/` - Baby Red Dragon (43 native PNGs + 43 4x PNGs)
- `emperor_penguin_pixel_pet_pack/` - Chubby Emperor Penguin (43 native PNGs + 43 4x PNGs)
- `arctic_fox_pixel_pet_pack/` - Arctic Fox (43 native PNGs + 43 4x PNGs)
- `robot_dog_pixel_pet_pack/` - Robot Dog (43 native PNGs + 43 4x PNGs)
- `boxer_dog_pixel_pet_pack/` - Boxer Dog (43 native PNGs + 43 4x PNGs)
- `bearded_dragon_pixel_pet_pack/` - Bearded Dragon (43 native PNGs + 43 4x PNGs)
